# 📰 Posting Page

## 🧩 Sobre o projeto

O **Posting Page** é uma aplicação web desenvolvida como projeto do curso **DEVstart** com foco no consumo e envio de dados para uma API externa. O usuário preenche um formulário com título e corpo de um post, que é então enviado para a [API pública JSONPlaceholder](https://jsonplaceholder.typicode.com/) via requisição POST.

A interface é simples e objetiva, apresentando ao usuário a lista de posts retornados pela API diretamente na tela.

---

## 🎯 Objetivo do projeto

O projeto tem como foco:

- Praticar o consumo de APIs REST com JavaScript puro
- Realizar envio de dados via método POST usando `fetch`
- Exibir dinamicamente dados retornados pela API no DOM
- Trabalhar com formulários HTML e captura de eventos

---

## 🚀 Tecnologias utilizadas

- **HTML5** — estrutura da página
- **CSS3** — estilização da interface
- **JavaScript** — lógica, manipulação do DOM e requisições à API

## ⚙️ Como executar o projeto

Clone este repositório:

```bash
git clone https://github.com/Caio-Dev0/posting--page
```

Abra o arquivo `index.html` diretamente no navegador — não é necessário nenhum servidor ou instalação de dependências, pois o projeto utiliza apenas HTML, CSS e JavaScript puro.

- ✅ Envio de dados para a API via método **POST**
- ✅ Integração com a API pública [JSONPlaceholder](https://jsonplaceholder.typicode.com/)
